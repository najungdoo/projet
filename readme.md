# 학생 QR 출결 관리 앱

학생마다 고유 QR 코드를 생성해 배부하고, 카메라 스캔으로 출결을 기록할 수 있는 **정적 웹 앱**입니다.

## 주요 기능

- 학생 목록 일괄 등록 (`학번,이름` 또는 `이름` 단독 입력)
- 학생별 QR 코드 생성 및 개별 PNG 다운로드
- 생성된 전체 QR 코드 인쇄 전용 화면 제공
- 카메라 기반 QR 스캔으로 당일 출석 체크
- 중복 스캔 방지(이미 출석 처리된 학생 안내)
- 출결 현황(출석/미출석, 체크 시간) 표시
- 당일 출결 CSV 다운로드 (`attendance-YYYY-MM-DD.csv`)
- 브라우저 `localStorage` 기반 데이터 영속화
- 3개 탭(생성 / 스캔 / 현황) 기반 UI 전환

## 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript
- [qrcode](https://www.npmjs.com/package/qrcode) (QR 생성)
- [html5-qrcode](https://github.com/mebjas/html5-qrcode) (카메라 스캔)

## 실행 방법

별도 빌드 없이 실행 가능합니다.

### 1) 파일 직접 열기

1. `index.html`을 브라우저에서 엽니다.
2. 학생 정보를 입력하고 **QR 생성**을 클릭합니다.
3. **스캐너 시작**으로 QR을 스캔해 출결을 기록합니다.
4. **출결 현황** 탭에서 결과 확인 후 CSV를 내려받습니다.

### 2) 로컬 서버 실행 (권장)

카메라 권한/브라우저 정책 이슈를 줄이려면 `localhost`로 실행하세요.

```bash
python3 -m http.server 8080
```

브라우저에서 `http://localhost:8080` 접속 후 사용합니다.

> 카메라 접근 권한 허용이 필요합니다.

## 사용 흐름

### 1. QR 생성 탭

- 입력 형식
  - `2026001,김민준`
  - `2026002,이서연`
  - `박지훈` (학번 미입력 시 자동 생성)
- **QR 생성** 클릭 시 학생 테이블과 QR이 생성됩니다.
- **QR 인쇄** 클릭 시 인쇄 전용 팝업이 열립니다.
- 각 학생 행의 **다운로드** 버튼으로 QR 이미지를 저장할 수 있습니다.

### 2. 출결 스캔 탭

- **스캐너 시작** 클릭 후 학생 QR을 카메라에 비춥니다.
- 유효한 학생 QR이면 즉시 출석 처리됩니다.
- 같은 학생을 다시 스캔하면 이미 처리되었음을 안내합니다.
- **오늘 출결 초기화**로 당일 출결만 리셋할 수 있습니다.

### 3. 출결 현황 탭

- 당일 기준 출석 요약(`출석 n명 / 총 n명`)을 표시합니다.
- 학생별 상태(출석/미출석)와 체크 시간을 확인할 수 있습니다.
- **CSV 다운로드**로 당일 기록을 파일로 저장할 수 있습니다.

## 데이터 저장 구조

브라우저 `localStorage`에 다음 키로 저장됩니다.

- `qr_attendance_students_v1`: 학생 배열
- `qr_attendance_records_v1`: 날짜별 출결 객체

출결 데이터는 `YYYY-MM-DD` 날짜 키 하위에 학번별 ISO 시간값으로 저장됩니다.

## 파일 구조

- `index.html`: 화면 구조 및 탭 구성
- `styles.css`: 레이아웃/컴포넌트 스타일
- `app.js`: 학생 파싱, QR 생성/스캔, 출결 및 CSV 처리 로직

## 빠른 점검 명령어

```bash
node --check app.js
```

문법 오류가 없으면 출력 없이 종료됩니다.
